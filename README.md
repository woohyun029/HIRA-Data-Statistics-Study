# 2023년 건강보험 시군구별 진료비 분석

> 건강보험심사평가원(HIRA) 공공데이터를 활용한 통계 학습 프로젝트입니다.  
> Python과 SAS를 병행하며 데이터 전처리부터 통계 검정, 회귀 모델링까지 전 과정을 직접 실습했습니다.

---

## 프로젝트 개요

### 데이터
- **출처**: 공공데이터포털 — 건강보험심사평가원 의료기관 시군구별 진료비 통계
- **기간**: 2023년 연간 집계 (단일 연도)
- **단위**: 전국 시군구 251개 (세종시 1개 포함)
- **주요 변수**: 환자수, 명세서청구건수, 입내원일수, 보험자부담금, 요양급여비용총액

### 분석 목적

특정 가설을 검증하기 위한 연구가 아니라, 통계 분석 전 과정을 스스로 익히기 위한 **학습 목적 프로젝트**입니다.

## 분석 단계

### 1단계 · 데이터 전처리

```
행 251개 · 열 8개 · 결측치 0개 · 중복 0개
```

원본 데이터는 결측치와 중복이 없어 정제 부담이 낮았습니다.
분석에 필요한 파생변수를 생성했습니다.

| 파생변수 | 산출 방법 |
|:---|:---|
| 1인당_진료비_만원 | 요양급여비용총액 ÷ 환자수 ÷ 10,000 |
| 1인당_내원일수 | 입내원일수 ÷ 환자수 |
| 보험자부담률 | 보험자부담금 ÷ 요양급여비용총액 × 100 |
| log_진료비 | log(1 + 1인당_진료비_만원) |
| log_환자수 | log(1 + 환자수) |
| 광역시 더미 | 서울·부산·대구·인천·광주·대전·울산·세종 = 1 |
| 강원더미 | 강원 = 1 |

### 2단계 · 탐색적 데이터 분석 (EDA)

**전국 요약**

| 지표 | 값 |
|:---|:---:|
| 전체 시군구 수 | 251개 |
| 1인당 진료비 평균 | 75.0만원 |
| 1인당 진료비 중앙값 | 73.1만원 |
| 최솟값 | 16.5만원 (인천 옹진군) |
| 최댓값 | 280.6만원 (전남 화순군) |
| 분포 왜도 | 1.619 (오른쪽 꼬리) |

**시도별 1인당 진료비 중앙값 (상위 3 / 하위 3)**

| 순위 | 시도 | 중앙값 |
|:---:|:---|:---:|
| 1 | 전북 | 96.8만원 |
| 2 | 광주 | 89.0만원 |
| 3 | 부산 | 84.4만원 |
| ⋮ | ⋮ | ⋮ |
| 15 | 충북 | 63.6만원 |
| 16 | 경북 | 62.4만원 |
| 17 | 강원 | 45.5만원 |

> 총액 기준으로는 서울이 압도적으로 크지만, 1인당 기준으로는 서울이 중간 수준(6위)입니다.  
> 화순군(280.6만원)이 최고값인 것은 화순전남대병원 등 지역 거점 병원의 영향으로 추정됩니다.  
> 단, 이는 데이터에서 관찰된 패턴이며 인과관계를 주장하는 것은 아닙니다.

### 3단계 · 통계 검정

정규성 검정 결과를 바탕으로 비모수 검정 방법을 선택했습니다.

#### 검정 1 — 정규성 (Shapiro-Wilk)

| 항목 | 원본 | 로그 변환 후 |
|:---|:---:|:---:|
| 왜도 | 1.619 | -0.512 |
| Shapiro-Wilk p값 | < 0.001 | < 0.001 |
| 판정 | ❌ 정규성 기각 | ❌ 기각 (개선됨) |

→ 이후 모든 그룹 비교는 **비모수 검정** 사용

#### 검정 2 — Mann-Whitney U (광역시 vs 비광역시)

| 그룹 | n | 중앙값 |
|:---|:---:|:---:|
| 광역시 | 76 | 81.0만원 |
| 비광역시 | 175 | 67.8만원 |

U = 8,176 · **p = 0.004** · 효과 크기 r = 0.18 (소~중간)

> 통계적으로 유의한 차이가 있으나, 효과 크기 r = 0.18은 크지 않습니다.  
> 두 그룹의 분포가 상당 부분 겹쳐 있어 광역시 여부만으로 진료비를 예측하기는 어렵습니다.

#### 검정 3 — Kruskal-Wallis (17개 시도)

H = 30.885 · **p = 0.014** · 효과 크기 η² = 0.064 (중간)

> 17개 시도 간 유의한 차이가 있으나, 시도 구분 자체는 진료비 변동의 6.4%만 설명합니다.

#### 검정 4 — Dunn 사후검정 (Bonferroni 보정)

전체 136쌍 비교 중 **유의한 쌍: 3개** (모두 강원도 포함)

| 비교 쌍 | p값 (Bonferroni) |
|:---|:---:|
| 강원 vs 서울 | 0.018 |
| 강원 vs 경남 | 0.046 |
| 강원 vs 부산 | 0.048 |

> 나머지 133쌍은 통계적으로 유의하지 않았습니다.  
> 강원도를 제외한 나머지 시도 간에는 보정 후 차이가 확인되지 않았습니다.

#### 검정 5 — Spearman 상관분석

| 변수 | ρ | p값 |
|:---|:---:|:---:|
| 환자수 ↔ 1인당 진료비 | 0.501 | < 0.001 |
| 1인당_내원일수 ↔ 1인당 진료비 | 0.345 | < 0.001 |
| 보험자부담률 ↔ 1인당 진료비 | 0.272 | < 0.001 |

---

### 4단계 · 모델링

#### Model 01 — 단순 선형회귀

Y: log(1인당_진료비), X: 1인당_내원일수

```
log(1인당_진료비) = 3.610 + 0.0547 × 1인당_내원일수
```

| 지표 | 값 |
|:---|:---:|
| R² | 0.151 |
| β₁ p값 | < 0.001 |
| AIC | 220.7 |
| 잔차 정규성 (SW p) | 0.100 ✅ |
| 등분산성 (BP p) | < 0.001 ⚠ |

내원일수 1일 증가 시 1인당 진료비 약 5.6% 증가 (단, 다른 변수 미통제 상태)

#### Model 02 — 다중 선형회귀

Y: log(1인당_진료비), X: 내원일수 · log(환자수) · 광역시 · 강원더미

| 변수 | β | 진료비 변화 | p값 | 유의 |
|:---|:---:|:---:|:---:|:---:|
| 1인당_내원일수 | 0.0893 | +9.3%/일 | < 0.001 | ✅ |
| log_환자수 | 0.2311 | +26.0% | < 0.001 | ✅ |
| 광역시 더미 | 0.0663 | +6.9% | 0.102 | ❌ |
| 강원 더미 | -0.0259 | -2.6% | 0.685 | ❌ |

| 지표 | Model 01 | Model 02 |
|:---|:---:|:---:|
| R² | 0.151 | **0.648** |
| AIC | 220.7 | **5.6** |
| 잔차 정규성 | ✅ 충족 | ❌ 위반 |
| VIF (최대) | — | 1.48 ✅ |

> 광역시·강원더미는 내원일수·환자수를 통제하면 통계적으로 유의하지 않았습니다.  
> 이는 지역 더미의 효과를 의료 이용량 변수가 매개할 가능성을 시사하지만,  
> 인과관계를 주장하기 위해서는 추가 분석이 필요합니다.  
>
> 잔차 정규성 위반(Shapiro-Wilk p ≈ 0)이 확인되어 HC3 강건 표준오차를 적용했습니다.  
> HC3 적용 후에도 유의 변수의 결론은 동일합니다.

#### Model 03 — 로지스틱 회귀

Y: 고진료비 여부 (1인당 진료비 > Q3 = 89.2만원 → 1)

| 변수 | OR | 95% CI | p값 | 유의 |
|:---|:---:|:---:|:---:|:---:|
| 1인당_내원일수 | 1.800 | [1.480, 2.191] | < 0.001 | ✅ |
| log_환자수 | 6.973 | [3.770, 12.899] | < 0.001 | ✅ |
| 광역시 더미 | 1.058 | [0.464, 2.409] | 0.894 | ❌ |
| 강원 더미 | 4.811 | [0.604, 38.302] | 0.138 | ❌ |

| 성능 지표 | 값 |
|:---|:---:|
| AUC | **0.844** |
| 정확도 | 80.1% |
| 고진료비 재현율 | 49.2% |
| Pseudo R² | 0.294 |

> 고진료비 시군구의 재현율이 49.2%로, 실제 고진료비 시군구 절반 가량을  
> 일반으로 잘못 분류합니다. 분류 목적에 따라 임계값 조정이 필요합니다.

#### 모델 비교 요약

| | Model 01 | Model 02 | Model 03 |
|:---|:---:|:---:|:---:|
| 목적 | 탐색 | 설명력 최대화 | 분류 |
| 종속변수 | 연속형 | 연속형 | 이진형 |
| 핵심 지표 | R²=0.151 | R²=0.648 | AUC=0.844 |
| 유의 변수 | 내원일수 | 내원일수·환자수 | 내원일수·환자수 |

## 주요 관찰 사항

아래는 이번 분석에서 일관되게 확인된 패턴입니다.  
**관찰된 데이터에서의 패턴이며, 인과관계에 대한 주장은 아닙니다.**

- 내원일수와 환자수 규모가 1인당 진료비와 통계적으로 유의한 양의 상관을 보였습니다
- 광역시·강원도 더미는 내원일수·환자수를 함께 투입했을 때 통계적 유의성이 사라졌습니다
- Model 02의 R² = 0.648은 진료비 변동의 약 35%가 이 모델로 설명되지 않음을 의미합니다
